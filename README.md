# Data_Analysis_Seoul_Hospital

이 저장소는 부스트코스에서 파이썬으로 시작하는 데이터 사이언스 강의[박조은](https://github.com/corazzon)를 실습한 내용입니다.

아나콘다 환경에서 파이썬3과 주피터노트북을 이용합니다.

사용한 라이브러리는 다음과 같습니다.
- numpy
- pandas
- matplotlib.pyplot
- seaborn
- folium

실습한 내용은 다음과 같습니다.

- 결측치 찾고 제거하기
- 수치 데이터 요약하기
- 문자열 데이터 요약하기
- 특정 column 별로 개수 세기,
- 위도,경도를 scatterplot로 표현 (seaborn)
- 위,경도와 주소데이터로 지도에 표현하기  (folium)
